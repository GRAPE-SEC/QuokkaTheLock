# 😈 Lock - (8/10 랜섬웨어 아이디어 피칭 정리본)

## Features
### Key 🔑
- **Steganography**: Key를 텍스트 안에 숨기거나 이미지로 변환 (게임 내 특정 단계에 도달 할 때까지 이미지 파일 detection 불가)
- 하이브리드 암호화 기능
### Logic Bomb / Timer 💣
- Logic Bomb: 특정 조건이 만족되었을 때 작동하는 기능
### Level System 🆙
- 틀리면 레벨 상향 or 클리어가 더 어려운 게임으로 종목 변환
### Anti-reversing 🍪
- 쿠키 키 쪼개기
